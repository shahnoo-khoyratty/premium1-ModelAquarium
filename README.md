\# Premium1 Model Aquarium — Static GitHub Pages
 


Static conversion of the WordPress one-page aquarium theme.



\## Structure

\- index.html — main one-page site

\- 404.html — not found page

\- assets/css/main.css — styles (from WordPress)

\- assets/js/main.js — scripts (from WordPress)

\- assets/images/ — site images



\## Deploy on GitHub Pages

1\. Create a repo and push this folder.

2\. In Settings → Pages, set Source to `main` branch, root.

3\. Visit the published URL.
